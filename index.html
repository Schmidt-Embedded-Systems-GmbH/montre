Montre

A timed regular expression matcher over timed symbolic behaviors

View the Project on GitHub doganulus/montre

  • Download ZIP File
  • Download TAR Ball
  • View On GitHub

Use for

  • Pattern search/extraction over timed behaviors (analog signals, execution traces, music, etc.)
  • Runtime monitoring/verification of real-time systems
  • Feature extraction and dimensionality reduction before temporal pattern mining

Get Started

First install Pure 0.64 and dependencies. For more details please refer to the original documents: https://bitbucket.org/purelang/pure-lang/wiki/GettingStarted

wget https://bitbucket.org/purelang/pure-lang/downloads/pure-0.64.tar.gz
tar xfvz pure-0.64.tar.gz
cd pure-0.64
./configure --enable-release
make
make check
sudo make install

Then install Montre as follows:

git clone https://github.com/doganulus/montre
cd montre
make
sudo make install
